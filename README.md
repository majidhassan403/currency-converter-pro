Yeh ek professional aur responsive Currency Converter hai jo live exchange rates fetch karta hai.

## 🚀 Features
- **Live Rates:** `open.er-api.com` se real-time data fetch karta hai.
- **Dynamic Flags:** Currency select karte hi flags khud-ba-khud update hote hain.
- **Fully Responsive:** Mobile aur Desktop dono par perfect chalta hai.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3
- **Logic:** Vanilla JavaScript (ES6+)
- **API:** ExchangeRate-API
